# 探针模式速查

## 代码调试

| 问题 | 探针 |
|---|---|
| 偶发 bug | 加日志，记录输入、环境和状态 |
| 性能瓶颈 | 写基准测试，只改一个变量 |
| 测试失败 | 最小复现，删到只剩失败路径 |
| 兼容性问题 | 固定浏览器/系统/版本矩阵 |
| 重构风险 | 加 characterization tests |

## 产品设计

| 问题 | 探针 |
|---|---|
| 需求不确定 | 访谈 5 个目标用户 |
| 方案争议 | 做两个低保真版本 |
| 指标卡住 | 拆漏斗，找最大流失点 |
| 文案不清 | A/B 两个首屏表达 |

## 写作与表达

| 问题 | 探针 |
|---|---|
| 不知从何写 | 先写 5 个论点标题 |
| 结构散 | 用一句话概括每段功能 |
| 没说服力 | 找一条反方意见回应 |

## 四选一

- 继续：探针支持假说。
- 转向：探针证伪假说。
- 回滚：探针揭示风险。
- 升级：需要权限、专家或资源。
