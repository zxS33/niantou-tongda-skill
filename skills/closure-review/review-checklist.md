# 收念闭环检查清单

## 事实

- [ ] 我是否只写结果，没有人格评价？
- [ ] 有没有补充日志、截图、测试或用户反馈？
- [ ] 还有哪些事实缺口？

## 责任

- [ ] 哪些是我可控的？
- [ ] 哪些是他人或系统因素？
- [ ] 哪些是不可预测环境？

## 原则

- [ ] 下次能实际改变哪一个行为？
- [ ] 原则是否具体可观察？
- [ ] 是否只保留一条核心原则？

## 残余动作

- [ ] 是否需要修复？
- [ ] 是否需要道歉或说明？
- [ ] 是否需要补文档、测试、监控？
- [ ] 是否需要关闭任务或通知相关方？

## 停止规则

- [ ] 没有新事实时是否停止复盘？
- [ ] 是否把注意力转回下一步？
