# 可控边界矩阵

| 圈层 | 问题 | 可用动作 |
|---|---|---|
| 控制圈 | 我今天能直接改变什么 | 写测试、补文档、发说明、做原型、拆任务 |
| 影响圈 | 我能提高什么概率 | 同步风险、给选项、拿数据、约评审、给 deadline |
| 观察圈 | 我只能等待什么信号 | 审批、市场反应、他人排期、政策变化 |

## 等待上限模板

```text
如果在 [时间] 前没有 [信号]，我将执行 [降级/替代/升级]。
```

## 边界话术模板

```text
我可以在本轮完成 A 和 B。C 会影响交付质量，需要二选一：延后 C，或调整截止时间。
```
