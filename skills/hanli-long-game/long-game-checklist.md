# 长线检查表

行动前检查：

- [ ] 这一步是否关闭关键退路？
- [ ] 证据是否足以支撑行动强度？
- [ ] 是否为了证明自己而不是服务目标？
- [ ] 是否增加长期维护债？
- [ ] 三个月后我是否仍认可这个选择？
- [ ] 有没有更小、更稳、更可逆的动作？

## 重手动作降级表

| 冲动动作 | 长线替代 |
|---|---|
| 重写整个模块 | 先加测试封住行为 |
| 激烈反驳 reviewer | 列事实和 tradeoff |
| 接下所有需求 | 给范围和排期选项 |
| 公开指责 | 先收集证据和私下沟通 |
| 推翻设计 | 做对照稿验证问题 |
